# RFM_IG_3_years

This repository contains the analysis for Instagram RFM/RCEDTG clustering across non-overlapping 3-year periods.

The code used for the analysis is available in the notebook:
- `non_overlapping_3y_periods_new_definitions.ipynb`

The folder also includes the following charts:
- Distribution plots of the RCEDTG variables, used to compare each variable across the 3-year periods.
- A bar chart to compare the clusters across the three 3-year periods.
- Pie charts to assess cluster composition in the 3-year periods.
- A radar chart to show which variables changed over time for each cluster.
