# RFM_IG_rolling_quadrimesters

This repository contains the analysis for Instagram RFM/RCEDTG clustering based on rolling quadrimesters.

The code used for the analysis is available in the notebook:
- `rolling_quadrimesters_new_definitions.ipynb`

The folder also includes the following charts:
- Distribution plots of the RCEDTG variables across the rolling quadrimester windows.
- K-means cluster visualizations for the clustering solutions.
- Transition matrices showing how users move across clusters over time.
- A Sankey diagram highlighting checkpoint flows across rolling quadrimesters.
- Summary tables as charts for state persistence, exit risk, and top exit destinations.
